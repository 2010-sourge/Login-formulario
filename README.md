# 🔐 Login Form

Uma página de login clean e moderna com fundo ciano/turquesa e textura de ondas, campos de autenticação em português e link para cadastro — desenvolvida com HTML e CSS puro.

---

## 📸 Preview

> Card de login centralizado sobre fundo ciano/turquesa com textura de raios irradiantes. Título **"Login Form"** em branco, campos de "Email ou Telefone" e "Senha" com ícones, link "Esqueceu a senha?", botão **Login** em ciano escuro e link "Não possui uma conta? **Cadastre-se agora**".

---

## 🚀 Tecnologias

- **HTML5**
- **CSS3**
- **Font Awesome** (ícones dos campos)

---

## 📁 Estrutura do Projeto

```
formulario-de-login/
├── index.html
└── style.css
```

---

## ⚙️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/formulario-de-login.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd formulario-de-login
   ```

3. Abra o `index.html` no navegador ou via **Live Server** no VS Code:
   - Acesse em: `http://127.0.0.1:5500/index.html`

---

## 🎯 Funcionalidades

- Campo de **Email ou Telefone** com ícone de usuário
- Campo de **Senha** com ícone de cadeado
- Link **"Esqueceu a senha?"** para recuperação
- Botão **Login** para autenticação
- Link **"Não possui uma conta? Cadastre-se agora"** para registro

---

## 🎨 Design

- Fundo: ciano/turquesa (`#00BCD4` aprox.) com textura de raios irradiantes
- Card: branco semitransparente com bordas arredondadas
- Título: branco centralizado em negrito
- Campos: fundo branco com ícone ciano à esquerda
- Botão Login: ciano escuro com texto branco
- Links: ciano com sublinhado

---

## 📌 Observações

- O projeto roda na porta `5500` via Live Server (VS Code)
- Título da aba: **Login Form**
- Pasta no Explorer: **FORMULARIO DE LOGIN** com `index.html` e `style.css`
- Interface totalmente em **português**
- Não requer dependências além do Font Awesome (via CDN)
